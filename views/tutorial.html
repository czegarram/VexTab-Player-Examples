Alternative Accidental Strategies

The new accidentals option has support for two strategies:

  • standard: Default strategy of displaying and cancelling accidentals based on key signature.
  • cautionary: Displays implied accidentals in parenthesis.

Cautionary Strategy

If you're using key/octave syntax (instead of the guitar-based fret/string syntax), you can suffix your accidental with a ~ to render it in parenthesis, e.g., A#/4 has no parenthesis, A#~/4 has parenthesis.

Jan 17 2013

Fret Hand Fingering and String Numbers

More love for classical instrumentalists. You can annotate your standard notation with string numbers and fret hand fingers using the $.fingering annotation. This is particularly useful for multi-voice notation.

The fingering annotation takes four parameters separated by colons (:):

  • Note number (The specific note number in the chord from bottom to top.)
  • Annotation position: a (Above), b (Below), l (Left), or r (Right)
  • f (Fret) or s (String)
  • a number to represent the fret or string.

For example notes (E/4.G/4.C/5.E/5) $.fingering/4:r:f:2.$ adds a fingering annotation (f) to the fourth note (4 which is the E). The annotation displays the number 2. You can add multiple fingerings to a note or chord by separating them with dashes -. (Click the Edit tab on the right to see some examples of this.)

Jan 15 2013

Playback Your Music!

Add options player=true to the beginning of your notation to add play and stop buttons. Then click play!

You can set the tempo with the tempo option. For example:

                                        vextab
                                        options player=true tempo=140
                                        tabstave notation=true tablature=false time=4/4
                                        notes :q (C/4.E/4.G/4) D-E-F/4

Note that there are still a few timing and lag issues due to the way playback is implemented. We are working on more robust playback support.

Playback works even when embedded into your own site. See this blogpost for a demo of an embedded player.

Add Sweep, Brush, or Rasquedo Strokes

With the $.stroke/TYPE.$ annotation, you can now notate different types of right hand strokes. The value for TYPE can be qu, qd, bu, bd, ru, rd, for rasquedo up/down, brush up/down, or roll up/down respectively.
